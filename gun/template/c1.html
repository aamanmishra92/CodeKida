General structure of C Program


      The structure of c program contains yhe following sections  :

  • Docement section
  • Pre Processor section
  • Global section
  • Main() section
  • User define section

1). Document section

  This section is related to user comment section.    

     Comments :  Comments are remark statement of a program that is used to describe the objective of program function or any statement. A good programmer always uses comments for better understanding of coad.

There are two type of comments :

1)Single line comments :   For single line comments with the help of "//" before a single line.



Example:  // This is single line comment.

1)Multiline comments :   For multiline comments with the help of "/* -----*/" before a and after the multiple line.



Example:  /* This is multiline comment.*/

2). Pre-processor section

Pre-processor section< is a section where number of header files are included , basicaly it helps to link some files with C program.

Header file :   The header files is collection of files and functions that are frequently used be C programs.


Every header files have some extension name.



Example:  #include "header file with extension"

3). Global section

This section is helpful to declare some variable which can access in all over program. Basically before main Function we can declare different global variable we can access those variable in all over program.

4). Main() section

All the C language codes are written within this section. This section contain two parts  :
a) Declaration part-  For declare variable.
b) Execution part-  All statement written between {-----} these braces are executed by C compiler.

5). User define section

Users can add number of different function with different datatype , user called some function number of times in main() function.
